## Hedef Metriklere Karsi Performans Durumu

| Metrik | Hedef | Gerceklesen | Durum |
|--------|-------|-------------|-------|
| CPC | 2-5 TL | | Hedefte / Altinda / Ustunde |
| CTR | >%3 | | Hedefte / Altinda / Ustunde |
| Donusum Orani | >%1 | | Hedefte / Altinda / Ustunde |

### Degerlendirme Kriterleri

- **CPC 2-5 TL arasi:** Normal. Bu araliktaki dalgalanmalar beklenir.
- **CPC 5 TL ustu:** Anahtar kelime rekabeti yuksek. Daha spesifik long-tail kelimelere yonelin veya reklam kalite puanini artirin.
- **CPC 2 TL alti:** Cok iyi. Bu kelimelere daha fazla butce yonlendirin.
- **CTR %3 alti:** Reklam metni cazip degil. Baslik ve aciklamalari test edin.
- **CTR %5 ustu:** Cok iyi performans. Bu reklam metnini diger gruplarda da deneyin.
- **Donusum orani %1 alti:** Acilis sayfasi (landing page) veya siparis sureci sorunlu olabilir. Siteyi kontrol edin.
