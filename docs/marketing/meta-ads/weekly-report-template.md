## Hedef Metriklere Karsi Performans Durumu

| Metrik | Hedef | Gerceklesen | Durum |
|--------|-------|-------------|-------|
| CPC | 1-3 TL | | Hedefte / Altinda / Ustunde |
| CTR | >%1 | | Hedefte / Altinda / Ustunde |
| Donusum Orani | >%1 | | Hedefte / Altinda / Ustunde |
| Sikilik | <3 | | Hedefte / Altinda / Ustunde |

### Degerlendirme Kriterleri

- **CPC 1-3 TL arasi:** Normal. Meta'da gorsel kalitesi CPC'yi dogrudan etkiler.
- **CPC 3 TL ustu:** Gorsel/video kalitesini artirin. Kitle hedeflemesini genisletin. Yeni reklam formati deneyin.
- **CPC 1 TL alti:** Cok iyi. Bu reklam setine butce artirin.
- **CTR %1 alti:** Gorsel dikkat cekici degil veya metin ilgi uyandirmiyor. Yeni yaratici icerik test edin.
- **CTR %3 ustu:** Mukemmel performans. Bu formati ve stili tekrarlayin.
- **Sikilik 3 ustu:** Kitle reklami cok fazla goruyor. Gorseli degistirin veya kitleyi genisletin.
- **Donusum orani %1 alti:** Siteye gelen trafik donusmuyor. Acilis sayfasini (landing page) kontrol edin. Mobil uyumluluk, yukleme hizi ve siparis surecini inceleyin.
